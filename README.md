# Login Form

एक सुन्दर र आधुनिक login form वेबसाइट। यसमा email र password validation छ।

## Features ✨

- ✅ Email validation
- ✅ Password validation (minimum 6 characters)
- ✅ Real-time form field validation
- ✅ Success/Error messages
- ✅ Responsive design (mobile-friendly)
- ✅ Beautiful gradient UI
- ✅ Smooth animations

## Files

- `index.html` - HTML structure
- `style.css` - Styling र animations
- `script.js` - JavaScript logic

## How to Use

1. Repository को files download गरु वा clone गरु
2. `index.html` file को खोलु browser मा
3. Login form मा email र password दर्ता गरु
4. Submit बटन क्लिक गरु

## Validation Rules

- **Email**: Valid email format आवश्यक छ (e.g., user@example.com)
- **Password**: कम्तिमा 6 characters आवश्यक छ

## Technologies Used

- HTML5
- CSS3 (Flexbox, Gradient, Animations)
- JavaScript (ES6)

## Browser Support

- Chrome ✅
- Firefox ✅
- Safari ✅
- Edge ✅
